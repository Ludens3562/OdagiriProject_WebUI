<b>プロジェクトサイト</b><br>
https://www.ne.senshu-u.ac.jp/~proj2023-05/

## 実行方法
① `docker-compose build`<br>
② `docker-compose up -d`<br>
③ `docker-compose run --rm web rails db:create`<br>

`localhost:3000` でWEBが立ち上がります。
